Detailed Description

Uebernimmt die TCP-Verbindung zum Client, veranlasst das zeitliche Voranschreiten aller Aufzugsystem-Komponenten, sendet dem Client die zur Visualisierung des Aufzugsystems notwendigen Daten und empfaengt Events vom Client, um diese weiterzuverarbeiten.
