Detailed Description

Modelliert eine Queue (Schlange) von Events. Die Controller hinterlegen hier alle Events, die von ihren angebundenen Sensoren und Aktoren empfangen werden. Es ist moeglich, neue Events zu hinterlegen (push) oder gespeicherte abzurufen und zu loeschen (pop).
